# Medication Donation Center Locator

A web application for finding and managing medication donation centers. Users can register, wait for admin approval, and view active donation centers. Admins can approve users, add donation centers, manage center status, and update timings.

## Live Demo

https://your-vercel-link.vercel.app

## Tech Stack

- React + Vite
- Django REST Framework
- MySQL
- Railway
- Vercel

## Features

- User registration and login
- Admin approval system
- Add donation centers
- Manage donation center status
- Update center timings
- View active donation centers
